еүҚдё–д»Ҡз”ҹ

SISD

иҝңеҸӨж—¶жңҹпјҢCPUжҳҜжҳҜSISDпјҲеҚ•жҢҮд»ӨпјҢеҚ•ж•°жҚ®пјү

Such CPUs were also scalar and integer in nature вҖ“ but what exactly does this mean? Scalar is where any math operation takes place on just single data values: itвҖҷs typically described as being SISD (single instruction, single data).

ж ҮйҮҸжҳҜд»Җд№Ҳж„ҸжҖқе‘ўпјҹе°ұжҳҜдёҖдёӘжҢҮд»ӨеӨ„зҗҶдёҖжқЎж•°жҚ®гҖӮжҜ”еҰӮпјҢжңүдёҖдёӘ32е…ғзҙ зҡ„ж•°з»„aпјҢз„¶еҗҺдҪ жғіз»ҷиҝҷдёӘж•°з»„дёӯзҡ„жҜҸдёӘе…ғзҙ йғҪеҠ 1пјҢйӮЈжҖҺд№ҲеҶҷе‘ўпјҢйӮЈе°ұеҫ—еҸ‘еҮә32ж¬ЎеҠ жҢҮд»ӨпјҢз„¶еҗҺеҺ»и®Ўз®—a[i] + 1гҖӮ

for (int i = 0; i < 32; ++i)
    a[i] += 1;

ж•ҙж•°еҸҲжҳҜд»Җд№Ҳж„ҸжҖқпјҹеӯ—йқўж„ҸжҖқпјҢе°ұжҳҜеҸӘиғҪеӨ„зҗҶж•ҙж•°пјҢд»ҘеүҚиҰҒжҳҜжғіеӨ„зҗҶжө®зӮ№ж•°зҡ„иҜқпјҢйңҖиҰҒеҸҰеӨ–зҡ„еҚ•зӢ¬зҡ„еӨ„зҗҶеҷЁд»ҘеҸҠеҚ•зӢ¬зҡ„жҢҮд»ӨйӣҶпјҲx87пјүгҖӮиҝҷдёӘеҺҶеҸІиҝҮзЁӢжҳҜиҝҷж ·зҡ„пјҡ

жңҖејҖе§Ӣе°ұжҳҜдё»жқҝдёҠйңҖиҰҒе®үиЈ…дёӨдёӘеӨ„зҗҶеҷЁпјҢдёҖдёӘдё“й—ЁеӨ„зҗҶж•ҙеһӢпјҢдёҖдёӘдё“й—ЁеӨ„зҗҶжө®зӮ№еһӢпјҢеҸ«еҚҸеӨ„зҗҶеҷЁ

еҗҺжқҘпјҢintel 80486еҸ‘еёғдәҶпјҢжҳҜз¬¬дёҖж¬ҫжҠҠйӮЈиҝҷдҝ©дёңиҘҝйӣҶжҲҗеҲ°дёҖиө·зҡ„иҠҜзүҮпјҢеӨ„зҗҶжө®зӮ№ж•°зҡ„еҷЁд»¶еҸ«FPUпјҢе°ҒиЈ…еңЁдёҖдёӘиҠҜзүҮйҮҢе°ұжҸҗй«ҳдәҶиҝҗз®—йҖҹеәҰеҳӣпјҢйҖҡи®Ҝжӣҙеҝ«

еҶҚеҗҺжқҘпјҢintelжҺЁеҮәдәҶе…·жңүMMXжҠҖжңҜзҡ„еҘ”и…ҫcpu

SIMDдёҺMMX

MMXзҡ„еҮәзҺ°ж”№еҸҳдәҶд»Җд№Ҳе‘ўпјҹиғҪж”ҜжҢҒеҗ‘йҮҸиҝҗз®—дәҶпјҢд№ҹе°ұжҳҜж”ҜжҢҒSIMDпјҲеҚ•жҢҮд»ӨпјҢеӨҡж•°жҚ®пјүдәҶгҖӮжҖҺд№Ҳж”ҜжҢҒзҡ„е‘ўпјҹдёҠйқўжҸҗеҲ°зҡ„FPUдёӯзҡ„еҜ„еӯҳеҷЁжҳҜ64дҪҚзҡ„пјҢжүҖд»ҘеҸҜд»ҘеҗҢж—¶еӯҳж”ҫе’Ңи®Ўз®—2дёӘ32дҪҚ/4дёӘ16дҪҚ/8дёӘ8дҪҚпјҢз„¶еҗҺзі»з»ҹе°ұеҸҜд»ҘеңЁдёҖжқЎжҢҮд»ӨдёӯеҗҢж—¶еӨ„зҗҶиҝҷдәӣеҶ…е®№дәҶгҖӮпјҲдҪҶжҳҜиҝҷж—¶еҖҷеҠҹиғҪиҝҳжҳҜжңүйҷҗзҡ„пјҢз¬¬дёҖпјҢж”ҜжҢҒеҗ‘йҮҸиҝҳжҳҜеҸӘйҖӮз”ЁдәҺж•ҙж•°и®Ўз®—пјҢз¬¬дәҢпјҢеӣ дёәжҳҜдҪҝз”ЁдәҶFPUдёӯзҡ„еҜ„еӯҳеҷЁиҝӣиЎҢи®Ўз®—зҡ„пјҢжүҖд»ҘдҪҝз”ЁMMXжҢҮд»Өзҡ„ж—¶еҖҷдёҚиғҪеҗҢж—¶иҝӣиЎҢжө®зӮ№иҝҗз®—пјү

иҝҷеқ—е…¶е®һжңүдёӘз–‘й—®жқҘзҡ„пјҢйӮЈеҪ“ж—¶зҡ„32дҪҚзі»з»ҹдёӢпјҢйҖҡз”Ёзҡ„32дҪҚеҜ„еӯҳеҷЁзҗҶи®әдёҠдёҚд№ҹиғҪе®һзҺ°SIMDеҳӣпјҹ32дҪҚеҜ„еӯҳеҷЁзҗҶи®әдёҠжҳҜеҸҜд»ҘеҗҢж—¶ж”ҜжҢҒ2дёӘ16дҪҚ/4дёӘ8дҪҚиҝҗиЎҢзҡ„еҗ§пјҹзЎ®е®һпјҢжҲ‘д»¬жүӢеҠЁж“ҚдҪңеҸҜд»Ҙе®һзҺ°иҝҷж ·зҡ„ж•ҲжһңпјҢдҪҶжҳҜжҜ”иҫғйә»зғҰпјҢжҜ”еҰӮиҰҒеңЁдёҖдёӘ32дҪҚйҖҡз”ЁеҜ„еӯҳеҷЁдёҠиҝӣиЎҢ2дёӘ16дҪҚеҠ жі•пјҢе°ұйңҖиҰҒйҖҡиҝҮз§»дҪҚд№Ӣзұ»зҡ„ж“ҚдҪңе…ҲжҠҠдёӨдёӘж•°ж”ҫиҝӣдёҖдёӘеҜ„еӯҳеҷЁдёӯпјҢз„¶еҗҺиҝҗз®—е®Ңд№ӢеҗҺпјҢиҝҳйңҖиҰҒжҠҠдёӨдёӘж•°еӯ—жҸҗеҸ–еҲҶзҰ»еҮәжқҘгҖӮ

йӮЈе…¶е®һSIMDдҫқиө–зҡ„жҳҜд»Җд№Ҳе‘ўпјҹжҳҜдәәе®¶зҡ„зЎ¬д»¶ж”ҜжҢҒеҠ жү©еұ•зҡ„жҢҮд»ӨйӣҶгҖӮдҪҝз”Ёзӣёе…ізҡ„жҢҮд»ӨеҸҜд»ҘзӣҙжҺҘиҝӣиЎҢSIMDзҡ„ж“ҚдҪңгҖӮ

SSE

еҶҚеҗҺжқҘе•ҠпјҢе°ұеҸҲеҮәзҺ°дәҶSSEгҖӮSSEеҸҲжңүе“Әдәӣж”№еҸҳе‘ўпјҹиҝҷдёҖж¬Ўе®ғдёҚеҶҚдҫқиө–FPUдёӯзҡ„64дҪҚеҜ„еӯҳеҷЁжқҘе®һзҺ°еҗ‘йҮҸиҝҗз®—дәҶпјҢиҖҢжҳҜеҚ•зӢ¬жү©еұ•еӨ„зҗҶ8дёӘ128дҪҚеҜ„еӯҳеҷЁпјҢиҝҳжү©еұ•дәҶдёҖдәӣиғҪеӨ„зҗҶжө®зӮ№ж•°зҡ„жҢҮд»ӨгҖӮ

дҪҶиҝҷж—¶еҖҷиҝҳжңүдёҖдәӣйҷҗеҲ¶пјҡSSEиҝҷж—¶еҖҷиҝҳеҸӘж”ҜжҢҒ32дҪҚзҡ„жө®зӮ№ж•°пјҢ4дёӘ32дҪҚжө®зӮ№ж•°пјҢдёҚж”ҜжҢҒж•ҙж•°иҝҗз®—пјӣиҖҢдё”е‘ўпјҢиҝҷж—¶еҖҷзҡ„SSEж”ҜжҢҒзҡ„128дҪҚеҜ„еӯҳеҷЁиҷҪз„¶дёҺFPUзҡ„еҜ„еӯҳеҷЁеҲҶејҖдәҶпјҢдҪҶжҳҜSSEжҢҮд»Өе’ҢFPUжҢҮд»Өе…¶е®һиҝҳжҳҜдёҚиғҪеҗҢж—¶иҝҗиЎҢзҡ„пјҢеӣ дёәеҸҜиғҪж¶үеҸҠеҲ°жҹҗдәӣзЎ¬д»¶иө„жәҗзҡ„е…ұдә«еҗ§гҖӮ

SSE2

еҗҺжқҘеҸҲеҚҮзә§дәҶSSE2пјҢиҝҷж—¶еҖҷж”ҜжҢҒзҡ„ж•°жҚ®зұ»еһӢе°ұжӣҙеӨҡдәҶгҖӮfour 32-bit or two 64-bit floats, as well as sixteen 8-bit, eight 16-bit, four 32-bit, or two 64-bit integers.

MMXеҜ„еӯҳеҷЁиҝҳдҝқз•ҷеңЁcpuдёӯпјҢдҪҶжҳҜжүҖжңүзҡ„MMXе’ҢSSEж“ҚдҪңйғҪеҸҜд»ҘдҪҝз”ЁеҚ•зӢ¬зҡ„128дҪҚSSEеҜ„еӯҳеҷЁиҝӣиЎҢеӯҳеӮЁгҖӮд»Җд№Ҳж„ҸжҖқе‘ўпјҹе°ұжҳҜе…¶е®һ128дҪҚзҡ„SSEеҜ„еӯҳеҷЁд№ҹеҸҜд»Ҙз”ЁжқҘиҝӣиЎҢMMXзҡ„ж“ҚдҪңпјҹ

SSE3

The MMX registers remained in the processor, but all MMX and SSE operations could be made to place using the separate 128-bit SSE registers.

SSE4

иҝҷдёӘзүҲжң¬дёӯпјҢеҜ„еӯҳеҷЁжІЎжңүдҝ®ж”№пјҢдҪҶжҳҜеўһеҠ дәҶжӣҙеӨҡжҢҮд»ӨиғҪеӨ„зҗҶжӣҙеӨҡеҸҜиғҪзҡ„ж•°еӯҰиҝҗз®—д№Ӣзұ»зҡ„пјҢд№ҹе°ұжҳҜжү©еұ•дәҶжҢҮд»ӨйӣҶ

AVXпјҲadvanced vector extensionsпјү

йҮҚеӨ§жӣҙж–°пјҢжҠҠеҗ‘йҮҸеҜ„еӯҳеҷЁзҡ„ж•°йҮҸжү©еұ•дәҶдёҖеҖҚпјҢеӨ§е°Ҹжү©еұ•дәҶдёҖеҖҚгҖӮд№ӢеүҚSSEж—¶д»ЈжҳҜ8дёӘ128дҪҚеҜ„еӯҳеҷЁпјҢзҺ°еңЁAVXж—¶д»Је·Із»Ҹжү©еұ•жҲҗдәҶ16дёӘ256дёӘеҜ„еӯҳеҷЁгҖӮ

дҪҶиҝҷж—¶еҖҷзҡ„ж•°жҚ®ж јејҸж–№йқўиҝҳдёҚеҰӮе·Із»ҸеҸ‘еұ•дәҶеҫҲд№…зҡ„SSE4пјҢзҺ°еңЁAVXеҸӘиғҪж”ҜжҢҒ8дёӘ32дҪҚжө®зӮ№/4дёӘ64дҪҚжө®зӮ№

зӣёеҜ№дәҺSSEпјҢAVXиҝҳжңүдёӘж”№еҸҳпјҢе°ұжҳҜжҸҗдҫӣдәҶдёҖж¬ЎеӨ„зҗҶдёүдёӘеҖјзҡ„иғҪеҠӣпјҡеңЁSSEдёӯпјҢжҜ”еҰӮжү§иЎҢе°ҶдёӨдёӘж•°зӣёеҠ зҡ„ж“ҚдҪңпјҢйӮЈзӣёеҠ зҡ„з»“жһңе…¶е®һжҳҜж”ҫеҲ°еӯҳж”ҫиҝҷдёӨдёӘеҠ ж•°дёӯзҡ„еҜ„еӯҳеҷЁзҡ„е…¶дёӯдёҖдёӘдёҠйқўзҡ„пјҢд№ҹе°ұжҳҜдјҡжҠҠжҹҗдёӘеҺҹе§Ӣзҡ„еҠ ж•°иҰҶзӣ–гҖӮиҖҢAVXдјҡдҝқжҠӨеҺҹе§ӢеҖјзҡ„е®үе…ЁпјҢжҠҠиҝҗз®—з»“жһңеӯҳеӮЁеҲ°еҚ•зӢ¬зҡ„еҜ„еӯҳеҷЁдёӯгҖӮпјҲе…¶е®һиҝҷеқ—жҲ‘жңүдёӘз–‘й—®пјҢдёҚзҹҘйҒ“д»–иҝҷдёӘжҢҮд»ӨйӣҶжҳҜжҖҺд№Ҳе®һзҺ°зҡ„пјҢеҰӮжһңжҠҠз»“жһңеӯҳеӮЁеҲ°еҲ«зҡ„еҚ•зӢ¬еҜ„еӯҳеҷЁдёӯпјҢжҳҜдёҚжҳҜйңҖиҰҒеҜ№йӮЈдёӘеҜ„еӯҳеҷЁдёӯзҡ„еҺҹе§Ӣзҡ„еҖјиҝӣиЎҢдҝқеӯҳе’ҢжҒўеӨҚе‘ўпјҹиҝҷж ·зҡ„иҜқжҳҜдёҚжҳҜжү§иЎҢдёҖжқЎжҢҮд»Өзҡ„ж—¶й—ҙе’ҢеӨҚжқӮеәҰиҰҒжҜ”SSEй«ҳпјҹпјү

Part of the problem of using AVX was that the load on the chip was so high that Intel had them automatically decrease the clocks in this mode, by around 20%, to keep the power consumption and heat levels down

иҝҷж—¶еҖҷйҷҚйў‘й—®йўҳе·Із»ҸеҲқзҺ°з«ҜеҖӘдәҶпјҢеӣ дёәеҜ„еӯҳеҷЁдҪҚж•°иҰҒжҜ”дёҖиҲ¬зҡ„32дҪҚжҲ–иҖ…64дҪҚеӨ§еҫҲеӨҡпјҢжүҖд»ҘеңЁиҝҗз®—ж—¶еҸҜиғҪи°ғз”ЁжӣҙеӨҡзҡ„жҷ¶дҪ“з®ЎпјҢеӣ жӯӨеҠҹиҖ—д№ҹжӣҙеӨ§пјҢиҖ—з”өеӨҡпјҢжүҖд»ҘйңҖиҰҒйҷҚйў‘

AVX2е’ҢFMA

еҠ дәҶж–°зҡ„жү©еұ•пјҢFMA(fused multiply-accumulate)пјҢдҪҶжҳҜжіЁж„ҸиҝҷжҳҜдёҖдёӘзӢ¬з«ӢдәҺAVXзҡ„еҠҹиғҪгҖӮ

FMAжҳҜд»Җд№Ҳпјҹе…Ғи®ёе°ҶеҠ жі•е’Ңд№ҳжі•еҗҲе№¶жҲҗдёҖжқЎжҢҮд»ӨпјҢжҜ”еҰӮFMAжҢҮд»ӨеҸҜд»ҘеңЁдёҖжқЎжҢҮд»ӨдёӯеҸ‘еҮәеҰӮдёӢж“ҚдҪң:FMA instruction(a, b, c) = a * b + cпјҢиҝҷдёӘж“ҚдҪңеңЁдёҖжқЎжҢҮд»Өдёӯе®ҢжҲҗгҖӮ

иҝҷеқ—зҡ„е®һзҺ°дёҠпјҢintelе’ҢamdиҝҳжңүзӮ№дёҚеҗҢе‘ўпјҡ

#include <stdio.h>
#include <immintrin.h>
#include <stdlib.h>
#include <time.h>

#define ARRAY_SIZE 1024 * 1024 * 16

int main() {
    float *a = (float *)_mm_malloc(ARRAY_SIZE * sizeof(float), 64);
    float *b = (float *)_mm_malloc(ARRAY_SIZE * sizeof(float), 64);
    float *c = (float *)_mm_malloc(ARRAY_SIZE * sizeof(float), 64);

    for (int i = 0; i < ARRAY_SIZE; i++) {
        a[i] = (float)rand() / RAND_MAX;
        b[i] = (float)rand() / RAND_MAX;
    }

    clock_t start = clock();

    int num_iterations = 1000;
    for (int iter = 0; iter < num_iterations; iter++) {
        for (int i = 0; i < ARRAY_SIZE; i += 16) {
            __m512 va = _mm512_load_ps(&a[i]);
            __m512 vb = _mm512_load_ps(&b[i]);
            __m512 vc = _mm512_add_ps(_mm512_mul_ps(va, vb), _mm512_set1_ps(1.0f));
            _mm512_store_ps(&c[i], vc);
        }
    }

    clock_t end = clock();
    double elapsed_time = (double)(end - start) / CLOCKS_PER_SEC;

    printf("AVX-512 и®Ўз®—е®ҢжҲҗпјҒ\n");
    printf("иҖ—ж—¶: %.6f з§’\n", elapsed_time);

    _mm_free(a);
    _mm_free(b);
    _mm_free(c);

    return 0;
}
